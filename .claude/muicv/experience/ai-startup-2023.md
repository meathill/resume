## 背景

- 2023 年 3 月，GPT-3.5 引爆 AI 应用浪潮之际，与资深 SEO 背景的朋友共同启动 AI 方向创业，使用合伙人已有公司 Vincent Co. 作为运营主体
- 作为联合创始人 / CTO，全面负责技术选型、产品研发、基础设施
- 2 年多期间连续探索多个产品方向，最终未能跑通商业化
- 2025 年 8 月团队和平解散，各自寻找新机会

## 尝试过的方向

1. **AI 写作辅助应用** —— AI 辅助内容创作方向
2. **AI 陪伴 / 心理疗愈应用** —— AI 情感陪伴与心理疗愈方向
3. **AI 翻译应用** —— 最接近成功的方向；积累了多位付费订阅用户，部分用户持续订阅数月；因创业末期资金与耐心见底，未能继续等到规模化

## 职责

- 作为 CTO 主导技术选型、架构设计与全栈落地
- 以 Vue/Nuxt 或 Next.js + LLM API 为核心，完成多次从 0 到 1 的产品搭建
- 使用 Vercel / Cloudflare / Docker 完成部署、CDN、边缘化交付
- 亲自参与产品打磨、用户运营与付费转化全流程

## 亮点

- **连续 3 个方向的完整 0→1 经验**：写作辅助 → 心理陪伴 → 翻译，覆盖工具、社交/健康、效率三大赛道
- **翻译产品实现真实付费订阅**：获取到持续订阅数月的付费用户，验证 AI 翻译方向的商业化可行性
- **在 2.5 年内以极小团队完成 3 次产品级迭代**：独立完成从 LLM 调用、前端、部署、订阅到用户增长的全栈闭环
- **站在 GPT-3.5 时代原点深度参与 AI 应用浪潮**，对 LLM 应用层的产品设计、成本控制、提示工程、用户留存有一手经验
- 坦诚面对创业结果：方向没跑通，但带走了一套可复制的「AI 应用独立开发 + 上线 + 变现」完整方法论
